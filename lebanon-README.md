# Lebanon D2P Training Website - Unit 2

## Overview
This website provides comprehensive information for the Lebanon Data to Policy (D2P) Training Program - Unit 2: Economic Evaluation of Policy, taking place June 15-18, 2026 in Amman, Jordan.

## Website Structure

### Main Pages:

1. **index.html** - Home Page
   - Program overview and timeline
   - Training objectives
   - Policy brief topics
   - Participant information

2. **agenda.html** - Training Agenda
   - Detailed 4-day schedule (June 15-18)
   - Module descriptions for each session
   - Learning objectives
   - Break times and logistics

3. **facilitators.html** - Facilitators & Mentors
   - Profiles of Chandra Dhakal, Rita Karam, Caroline Daccache
   - Mentor team structure
   - Group assignments
   - Contact information

4. **materials.html** - Training Materials (Primary Focus: Unit 2)
   - **Unit 2 materials** (featured prominently):
     - L09: Framing Economic Evaluation
     - L10: Assessing Health Impact
     - L11: Cost Analysis
     - L12: Economic Evaluation Methods
     - L13: Sensitivity Analysis
   - Economic tools and Excel templates
   - Case studies and examples
   - Activity sheets and worksheets
   - Unit 1 materials (reference)
   - Unit 3 materials (preview)
   - Lebanon-specific policy brief documents
   - Reference guides

5. **logistics.html** - Logistics & Practical Information
   - W Amman Hotel venue details
   - Travel information for Lebanese participants
   - Visa requirements
   - Accommodation options
   - Meals and refreshments
   - What to bring (emphasis on laptop with Excel)
   - Currency and expenses
   - Pre-departure checklist

## Key Features

### Lebanon Program Specifics:
- **Two Policy Brief Topics:**
  1. Strengthening HTA to Address Rising Oncology Costs
  2. Priority-Setting Mechanism for HTA in Lebanon

- **12 Participants** divided into 2 groups of 6

- **Dedicated Mentors** for each group

### Unit 2 Focus:
- Extensive economic evaluation materials
- Excel templates for decision trees, cost inventories, CEA, sensitivity analysis

## Setup Instructions

### For GitHub Pages:

1. **Create Repository:**
   ```
   Repository name: lebanon-d2p-unit2-2026
   Description: Lebanon D2P Training - Unit 2: Economic Evaluation
   Public repository
   ```

2. **Upload Files:**
   - Upload all HTML files
   - Upload lebanon-styles.css (rename to styles.css)
   - Upload README.md

3. **Enable GitHub Pages:**
   - Settings → Pages
   - Source: main branch, / (root)
   - Save

4. **Your URL will be:**
   ```
   https://[username].github.io/lebanon-d2p-unit2-2026/
   ```

## Customization Guide

### Adding Training Materials:

1. **Create Google Drive folders** for:
   - Unit 2 Lectures
   - Economic Tools
   - Case Studies
   - Unit 1 Materials (reference)
   - Unit 3 Materials (upcoming)

2. **Update links in materials.html:**
   - Find `href="#"` placeholders
   - Replace with Google Drive sharing links
   - Ensure permissions are set to "Anyone with link can view"

3. **Upload participant documents:**
   - Problem statements from Unit 1
   - Policy options tables
   - Stakeholder analyses
   - Working decision trees and cost inventories

### Updating Content:

**Facilitator photos:** Add photos to an `images/` folder and update facilitators.html:
```html
<img src="images/chandra.jpg" alt="Chandra Dhakal" class="facilitator-photo">
```

**Hotel details:** Update W Amman contact information in logistics.html

**Participant emails:** Add email addresses for Rita Karam and Caroline Daccache
